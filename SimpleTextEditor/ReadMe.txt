Text Editor----------
The Text Editor is a Java-based application that provides a user interface for creating, editing, and saving text documents. It offers essential functionality such as opening and saving files, as well as basic text editing operations like cut, copy, and paste.

Features-------------
User-friendly graphical interface built using Java Swing components.
Text area for entering and editing text.
File menu with options to open, save, and close files.
Edit menu with cut, copy, and paste functionality.
"Save and Submit" button to save the data and close the file.
Ability to open and save text files in various formats.
Basic error handling and file input/output operations.
Scrollable text area for comfortable viewing and editing of large text files.

Requirements------------
Java Development Kit (JDK) 8 or later
Java Swing library
